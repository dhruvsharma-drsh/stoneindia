# GWALIOR STONE — Design System Details

**Global Font Strategy**
*   The entire website relies on a single, unified system rounded font stack (`ui-rounded`, `SF Pro Rounded`, `system-ui`, etc.).
*   No external web fonts (like Google Fonts) are loaded, ensuring maximum performance and visual consistency.

**Global Brand Colors**
*   **Gold (`#B8955D`)** serves as the primary accent color for buttons, icons, highlights, and special text.
*   **Light Gold (`#D2AF75`)** is applied for hover states and subtle badges.

---

## Thematic Details Per Page

| Page | Theme | Key Features & Typography |
| :--- | :--- | :--- |
| **Homepage** | **Dark Theme** | Provides a cinematic, luxury feel. Highlighted with White and Gold typography. Includes very large responsive headings (up to 9xl) and character-by-character reveal animations. |
| **About Page** | **Light Theme** (Main) <br> **Dark Theme** (Hero) | Utilizes a clean Light Theme for the main story and stats sections to improve readability, contrasting with a Dark Theme overlay for the hero banner. Typography is predominantly dark and muted, with Gold for key statistics. |
| **Infrastructure Page** | **Light Theme** (Cards) <br> **Dark Theme** (Hero/Stats) | Features a Light Theme for the facility description cards, contrasting with a Dark Theme for both the hero section and the statistics bar at the bottom. |
| **Projects Page** | **Light Theme** | Uses a bright Light Theme across the page to ensure project images stand out clearly. Features interactive particle backgrounds, image selector galleries, and dark text with Gold accents. |
| **Packaging Page** | **Cream Theme** (Hero/Badges) <br> **Light Theme** (Cards) <br> **Dark Theme** (Process) | Presents a warm Cream Theme for the hero, transitions to a bright Light Theme for the packaging feature cards, and ends with a bold Dark Theme for the process section featuring very large Gold numbers. |
| **Blog Page** | **Cream Theme** (Main) <br> **Light Theme** (Featured) | Uses a warm Cream Theme for the main page and post grids, and a clean Light Theme for the featured post card to make it stand out. Typography incorporates deep green and Gold in the hero section. |
| **Blog Post Page** | **Cream Theme** (Article) <br> **Dark Theme** (Hero/Next) | Features a comfortable Cream Theme for the article reading area to reduce eye strain, bordered by a Dark Theme for the hero overlay and the "Next Article" section. Uses near-black and deep green for text and headings. |
| **Product Detail Page** | **Cream Theme** (Info) <br> **Dark Theme** (Hero/Trust) | Utilizes a warm Cream Theme for product information, specifications, and pattern displays, contrasting with a Dark Theme for the hero background and the "Trust" banner. Consists of dark text and Gold accents. |
| **Contact Page** | **Light Theme** | Uses a simple, clean Light Theme. Typography features near-black text for the main content and Gold for highlights and contact form elements. |
| **Global Navigation** | **Dark Theme** | Features a sleek Dark Theme with a background blur effect. Typography uses White for "GWALIOR" and Gold for "STONE", with translucent white links that turn Gold on hover. |
